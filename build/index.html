About

AWS certificated Software Engineer with a great interest in working with the latest web and Mobile App technologies, and Full-stack web developer with a passion for building web/app that helps people with burning problems. Over 8 years of experience with React, NextJS, Node, Laravel, PHP, Angular, React Native, and more. I love developing applications with beautiful UI/UX with NextJS and React. Currently, I am working on applications built on React with state management using MobX, but eventually, I love to be working with NextJS with GraphQL or even a REST API with React Query.


I believe that technology is the most effective when it empowers people to solve their true issues and problems. I am inspired by tech entrepreneurs who make it their life's mission to make other people's lives better through technology. This could be a big idea like going to Mars, or it could be a "minor" one like making the UX on a site more intuitive.


I have a strong focus on crafting accessible, user-friendly, and multi-functionally webs/apps that provide a tangible solution for the user.


In the past few years, I have concentrated on maximizing my learning, and I have the skills of the following:


* Frontend: React, JavaScript, NextJS, Angular, HTML, CSS

* Backend: Node (Express, Koa), Golang, Ruby, PHP, Laravel

* Mobile: React Native, Flutter, IOS, Android, Swift

* Databases: SQL, MongoDB, PostgreSQL

* CMS: WordPress, Shopify

* AWS certification

* CI/CD Test

* Frontend: React, JavaScript, NextJS, Angular, HTML, CSS


I enjoy building strong relationships and work well with people at all levels of the organization including management, staff, and customers.
